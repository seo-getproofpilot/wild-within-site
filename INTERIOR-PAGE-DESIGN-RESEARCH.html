1. Interior Page Headers

The biggest gap in most therapy sites (and what will make Wild Within stand out).

What the competitors do

  • Meaningful Connection: Plain H1 + H2 question. No background image, no visual treatment. Just green serif text on white. Boring.
  • SonderMind: Large serif heading with staggered fade-in animations. Background uses CSS gradient (linear-gradient(70deg, #045 50%, #41bac6)). Most elevated of the group.
  • Grow Therapy: Full-viewport hero with background image, centered heading at 3.5rem bold.
  • TherapyDen: Hero gradient from teal to light teal (linear-gradient(to bottom, #21ada1 10%, #80e7de 100%)). Clean and impactful.
What Wild Within already has: The page-header with floating blurred orbs is already more sophisticated than 4 out of 5 competitors. The orb approach (blurred circles with animation) is unique in this space.

Upgrade: Add a subtle background texture + gradient overlay to page headers

Steal from SonderMind's approach but keep Wild Within's earthy palette:

5. Decorative Elements Worth Stealing

A. SonderMind: Staggered Fade-In Animations

This is the single highest-impact upgrade. SonderMind sections fade in as you scroll, with staggered timing:

/* Scroll-triggered reveal animations (SonderMind pattern) */
.reveal {
  opacity: 0;
  transform: translateY(30px);
  transition: opacity 0.6s ease, transform 0.8s ease;
}

.reveal.visible {
  opacity: 1;
  transform: translateY(0);
}

/* Stagger children within a section */
.reveal-group .reveal:nth-child(1) { transition-delay: 0s; }
.reveal-group .reveal:nth-child(2) { transition-delay: 0.15s; }
.reveal-group .reveal:nth-child(3) { transition-delay: 0.3s; }
.reveal-group .reveal:nth-child(4) { transition-delay: 0.45s; }

/* JavaScript for Intersection Observer */
<script>
document.addEventListener('DOMContentLoaded', () => {
  const reveals = document.querySelectorAll('.reveal');
  const observer = new IntersectionObserver((entries) => {
    entries.forEach(entry => {
      if (entry.isIntersecting) {
        entry.target.classList.add('visible');
        observer.unobserve(entry.target);
      }
    });
  }, { threshold: 0.15, rootMargin: '0px 0px -50px 0px' });

  reveals.forEach(el => observer.observe(el));
});
</script>
